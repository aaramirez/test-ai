# Como Tomar un Tutorial

## Que Son los Tutoriales?

Los tutoriales son experiencias de aprendizaje interactivo con caminos ramificados, gamificacion (XP, rachas, logros) y seguimiento de progreso en tiempo real. Se adaptan a tus respuestas y ofrecen ejercicios practicos.

## Ejecutar un Tutorial

1. Abre opencode en el directorio del proyecto
2. Escribe `/tutorial`
3. El agente listara los tutoriales disponibles
4. Selecciona el tutorial que deseas tomar
5. Ingresa tu ID de participante (o registrate si es primera vez)
6. Ingresa tu nombre cuando se te solicite

## Tipos de Pasos

Los tutoriales tienen diferentes tipos de pasos:

| Tipo | Que Haces |
|------|-----------|
| **Contenido** | Lee y aprende — presiona Continuar cuando estes listo |
| **Pregunta** | Responde una pregunta de opcion multiple |
| **Opcion** | Toma una decision que ramifica el camino del tutorial |
| **Codigo** | Ejecuta un ejercicio de codigo en tu terminal |
| **Desafio** | Crea o edita un archivo de forma practica |
| **Escenario** | Lee una historia y toma una decision con retroalimentacion |
| **Punto de control** | Quiz de puerta — debes aprobar para continuar |

## Gamificacion

### Sistema de XP
- Respuestas correctas: +10 XP
- Ejercicios de codigo: +5 XP
- Desafios: +20 XP
- Bonificaciones de racha en 3, 5 y 10 respuestas correctas consecutivas

### Logros
| Logro | Como Obtenerlo |
|-------|----------------|
| Primeros Pasos | Completa tu primer tutorial |
| Puntuacion Perfecta | Obtén 100% |
| En Llamas | Obtén una racha de 5+ correctas |
| Corredor de Codigo | Ejecuta todos los ejercicios de codigo |
| Aprendiz Rapido | Completa en menos de 5 minutos |
| Explorador | Completa 3+ tutoriales |

### Rachas
Las respuestas correctas consecutivas construyen una racha. Rachas mas altas generan bonificaciones de XP:
- 3 correctas: +5 XP bonus
- 5 correctas: +10 XP bonus
- 10 correctas: +25 XP bonus

## Consejos

- Lee los pasos de contenido cuidadosamente — las preguntas vienen despues
- Si fallas un punto de control, puedes reintentar el tutorial
- Los ejercicios de codigo son seguros — se ejecutan en un sandbox
- Prueba diferentes opciones para explorar caminos ramificados
- Revisa tus XP y logros al final de cada tutorial
