# How to Take a Tutorial

## What Are Tutorials?

Tutorials are interactive learning experiences with branching paths, gamification (XP, streaks, achievements), and real-time progress tracking. They adapt to your answers and provide hands-on exercises.

## Running a Tutorial

1. Open opencode in the project directory
2. Type `/tutorial`
3. The agent will list available tutorials
4. Select the tutorial you want to take
5. Enter your participant ID (or register if first time)
6. Enter your name when prompted

## Step Types

Tutorials have different step types:

| Type | What You Do |
|------|-------------|
| **Content** | Read and learn — press Continue when ready |
| **Question** | Answer a multiple-choice question |
| **Choice** | Make a decision that branches the tutorial path |
| **Code** | Run a code exercise in your terminal |
| **Challenge** | Create or edit a file hands-on |
| **Scenario** | Read a story and make a decision with feedback |
| **Checkpoint** | Gate quiz — must pass to continue |

## Gamification

### XP System
- Correct answers: +10 XP
- Code exercises: +5 XP
- Challenges: +20 XP
- Streak bonuses at 3, 5, and 10 consecutive correct answers

### Achievements
| Achievement | How to Earn |
|-------------|-------------|
| Primeros Pasos | Complete your first tutorial |
| Puntuacion Perfecta | Score 100% |
| En Llamas | Get a streak of 5+ correct |
| Corredor de Codigo | Run all code exercises |
| Aprendiz Rapido | Complete in under 5 minutes |
| Explorador | Complete 3+ tutorials |

### Streaks
Consecutive correct answers build a streak. Higher streaks earn bonus XP:
- 3 correct: +5 bonus XP
- 5 correct: +10 bonus XP
- 10 correct: +25 bonus XP

## Tips

- Read content steps carefully — questions follow from them
- If you fail a checkpoint, you can retry the tutorial
- Code exercises are safe — they run in a sandbox
- Try different choices to explore branching paths
- Check your XP and achievements at the end of each tutorial
